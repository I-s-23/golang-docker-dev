# golang-docker-dev
golang dev docker with VS Code extend remote-containers

## Version

- Golang
    - latest container
        - 1.16
- Docker

## Quick Start

1. VsCodeで`Ctrl + Shift + P`を押下
1. `Remote-Containers: Open Folder in Container`と入力
1. `golang-docker-dev`をCloneしたディレクトリを選択
2. Dockerが起動して、`golang`が使用可能になる
